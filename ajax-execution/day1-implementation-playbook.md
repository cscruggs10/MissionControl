## Success Indicators (First Week)

### Red Flags (means approach isn't working):
- [ ] 0 Loom views after 10 sent → personalization issue
- [ ] 0 email replies after 20 sent → subject lines or timing issue
- [ ] Loom views <30% → wrong audience or wrong angle
- [ ] Email open rate <20% → subject line problem

### Green Flags (means we're on the right track):
- [ ] Loom view rate 40%+ → people are interested
- [ ] Loom reply rate 2%+ → right audience + right message
- [ ] Email open rate 30%+ → good subject lines
- [ ] Email reply rate 1%+ → valuable content hitting target
- [ ] First Loom gets 3+ replies → messaging resonates
- [ ] Calendar booking rate 0.5%+ → convincing enough to book

### Optimization Triggers:
- If Loom views <40% → test different script angles
- If replies <2% → test different opening hooks
- If no meetings in first 20 contacts → CPA list may be wrong target
- If meetings booked but no partnerships → sales conversation issue (not outreach)

## Critical Success Factors

1. **Personalization > Template:** Each Loom must reference something specific. Each email must be customized. This is what drives 2-3x response rate.

2. **Week 1 = Validation Phase:** Don't scale until you've proven the approach works. Better to send 20 high-touch Looms + get 5 replies than 200 templated emails + get 2 replies.

3. **Messaging precision:** "Stop being commoditized" resonates with advisors. "Reduce your taxes" resonates with business owners. Getting the message right matters more than volume.

4. **Multi-touch wins:** Loom → email → call → letter. Sequence matters. Don't just email. Don't just call. Combination = magic.

5. **Data-driven iteration:** Track everything. If something isn't working, adjust and test again. Don't spray and pray.
